## 技术细节

### 数据库表
- `memos` - 备忘录
- `tasks` - 任务
- `mood_stats` - 心情统计
- `environment_stats` - 环境统计
- `diary_entries` - 日记条目（计划）
- `reminders` - 提醒（计划）
- `reports` - 报告（计划）

### 主要组件
- `HomeScreen` - 备忘录列表
- `MemoDetailScreen` - 备忘录详情
- `StatisticsScreen` - 统计分析
- `TagsEditor` - 标签编辑器
- `CelebrationOverlay` - 庆祝动画

### 工具函数
- `calculateSuggestedTime()` - 计算建议处理时间
- `formatDeadline()` - 格式化截止日期
- `exportToCSV()` - 导出为CSV
- `exportToJSON()` - 导出为JSON

### 如何发布新版本

1. 更新版本号在 `package.json`
2. 创建Git标签:
   ```bash
   git tag v1.0.1
   git push origin v1.0.1
   ```
3. GitHub Actions自动创建Release并上传构建包

访问 GitHub Releases 页面下载最新版本
