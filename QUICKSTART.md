# 快速开始指南

## 🎯 5分钟快速上手

### 第一步：安装依赖
```bash
npm install
```

### 第二步：启动开发服务器
```bash
npm run dev
```

应用会在浏览器中打开，或访问 `http://localhost:19006`

### 第三步：创建第一个备忘录
1. 点击右下方的 **+** 按钮
2. 输入备忘录名称（如：工作计划）
3. 选择优先级（低/中/高）
4. 可选：设置截止日期（YYYY-MM-DD）
5. 可选：添加标签（如：工作、紧急）
6. 点击"创建"按钮

### 第四步：添加任务
1. 进入备忘录详情页
2. 在输入框输入任务内容
3. 可选：点击图片按钮添加图片
4. 点击 + 按钮添加任务

### 第五步：完成任务
1. 点击任务前的复选框标记完成
2. 完成所有任务后，会看到庆祝动画 🎉
3. 可在任务后点击😊记录完成时的心情

## 🎮 主要功能快速导航

### 搜索备忘录
- 在顶部搜索框输入备忘录名称或标签
- 实时显示匹配的结果

### 编辑备忘录标签
- 点击备忘录卡片右侧的 🏷️ 按钮
- 选择或创建标签
- 点击"完成"保存

### 分享备忘录
- 点击备忘录卡片右侧的 📤 按钮
- 选择分享方式
- 包含所有任务和完成进度

### 导出数据
- 点击备忘录卡片右侧的 📥 按钮
- 选择导出格式（CSV 或 JSON）
- 自动下载文件

### 查看统计
- 点击底部导航"统计"标签
- 查看完整的数据分析和图表

### 删除备忘录
- 点击备忘录卡片右侧的 🗑️ 按钮
- 确认删除

## 🎨 界面说明

### 备忘录卡片
```
┌─ [优先级条] ─────────────────────────────┐
│ 📌 备忘录名称              ➜           │
│ 日期      建议处理时间    截止日期      │
│ 🏷️ 标签1  🏷️ 标签2                    │
│                              🏷️ 📤 📥 🗑️ │
└───────────────────────────────────────────┘
```

- **优先级条** - 左边框颜色：绿(低) 橙(中) 红(高)
- **日期** - 备忘录创建日期
- **建议处理时间** - 根据优先级和截止日期自动计算
- **截止日期** - 距离截止日期的天数
- **标签** - 绿色标签，可点击搜索
- **操作按钮**：
  - 🏷️ 编辑标签
  - 📤 分享备忘录
  - 📥 导出为 CSV/JSON
  - 🗑️ 删除备忘录

### 任务卡片
```
┌──────────────────────────────────────┐
│ ☐/☑️ ●●●  任务内容              🗑️  │
│       [任务图片]                  😊  │
│       🏠 😊                            │
└──────────────────────────────────────┘
```

- **复选框** - 点击标记完成/未完成
- **优先级点** - 颜色标识优先级
- **任务内容** - 已完成显示删除线
- **图片** - 可选的任务图片
- **环境标签** - 完成地点（家/办公室等）
- **心情** - 完成时的心情emoji
- **操作按钮**：
  - 😊 记录心情
  - 🗑️ 删除任务

## 📊 统计页面

### 总览卡片
显示：
- 备忘录总数
- 任务总数
- 已完成任务数
- 完成率百分比

### 完成情况
饼图展示已完成(绿)和未完成(橙)任务比例

### 每月统计
柱状图显示过去6个月的：
- 已完成任务(绿色柱)
- 总任务数(橙色柱)

### 备忘录排名
显示完成任务最多的前5个备忘录，包括：
- 排名号
- 备忘录名称
- 进度条
- 完成比例

### 工作环境分布
显示在不同地点完成的任务数量：
- 家
- 办公室
- 咖啡馆
- 其他

### 心情统计
显示不同心情下完成的任务数量：
- 😊 (开心)
- 😐 (平静)
- 😫 (难受)

## 🐛 常见问题

### Q: 如何恢复已删除的备忘录？
A: 删除是永久的，建议创建前使用导出功能备份数据。

### Q: 任务图片可以修改吗？
A: 目前需要删除后重新添加。任务编辑功能正在开发中。

### Q: 如何离线使用？
A: 当前版本需要网络连接。离线功能计划在未来版本中支持。

### Q: 可以导入旧数据吗？
A: 目前不支持导入，但可以通过导出功能备份数据。

### Q: 如何备份我的数据？
A: 使用导出功能将备忘录导出为JSON格式，保存本地文件。
